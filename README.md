# Salesforce UI Clone

A Salesforce-inspired homepage clone built using HTML and CSS.

## Overview

This project recreates the user interface of the Salesforce homepage to practice modern web development concepts such as layout design, navigation bars, hero sections, flexbox, and hover effects.

## Technologies Used

* HTML5
* CSS3
* Remix Icons

## Features

* Fixed Navigation Bar
* Hero Section
* Call-to-Action Buttons
* Hover Effects
* Flexbox Layout
* External Image Integration

## What I Learned

* HTML Page Structure
* CSS Styling
* Flexbox
* Positioning
* Hover Effects
* Building Real-World UI Layouts
